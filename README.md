# Starter point

This is based on SMACSS and BEM

**Includes**

- susy grid system
- compass


## File System

	--base
		_layout
		_config
		_typography

	--modules
		_buttons
		_lists
		_navigation

	--settings
		_extends
		_mixins
		_variables	

	--state
		_state

## Requirements

- Sass {http://sass-lang.com/}
- Compass {http://compass-style.org/install/}
- Susy {http://susy.oddbird.net/}

## Installing

1. `$ gem update --system`
2. `$ gem install susy`
