# Starter point

This is based on SMACSS and BEM

**Includes**

- susy grid system
- compass


## File Structure

	styles
	├── base
	│   ├── base.scss
	│   ├── layout.scss
	│   ├── config.scss
	│   ├── typography.scss
	├── modules
	│   ├── buttons.scss
	│   ├── list.scss
	│	└── navigation.scss
	├── settings
	│   ├── extend.scss
	│   ├── mixins.scss
	│	└── variables.scss
	└── state
	    └── state.scss

## Requirements

- Sass {http://sass-lang.com/}
- Compass {http://compass-style.org/install/}
- Susy {http://susy.oddbird.net/}

## Installing

1. `$ gem update --system`
2. `$ gem install susy`
